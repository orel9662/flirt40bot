# 🤖 בוט היכרויות - מדריך התקנה מלא

## שלב 1 - יצירת הבוט בטלגרם

1. פתח טלגרם וחפש `@BotFather`
2. שלח `/newbot`
3. בחר שם לבוט (לדוגמה: "בוט היכרויות")
4. בחר שם משתמש (חייב להסתיים ב-bot, לדוגמה: `my_dating_bot`)
5. **שמור את הטוקן שקיבלת!** (נראה כך: `123456:ABCdef...`)

## שלב 2 - מצא את ה-ID שלך בטלגרם

1. חפש `@userinfobot` בטלגרם
2. שלח `/start`
3. **שמור את המספר שמופיע ב-Id** (זה ה-ADMIN_ID שלך)

## שלב 3 - העלאה ל-Railway (חינמי)

1. היכנס ל: https://railway.app
2. לחץ "Login with GitHub" (צור חשבון GitHub אם אין לך - חינמי)
3. לחץ "New Project" → "Deploy from GitHub repo"
4. העלה את תיקיית הבוט ל-GitHub (ראה שלב 3א)
5. חבר את ה-repo ל-Railway

### שלב 3א - העלאה ל-GitHub

1. היכנס ל: https://github.com
2. לחץ "New repository"
3. קרא לו `dating-bot`
4. לחץ "Create repository"
5. גרור את כל קבצי הבוט לתוך ה-repo
6. לחץ "Commit changes"

## שלב 4 - הגדרת משתנים ב-Railway

ב-Railway, לך ל-Variables והוסף:
```
BOT_TOKEN = הטוקן שקיבלת מ-BotFather
ADMIN_ID = ה-ID שלך שקיבלת מ-userinfobot
```

## שלב 5 - הפעלה!

Railway יפעיל את הבוט אוטומטית. חפש את שם הבוט שלך בטלגרם ושלח `/start`!

---

## פקודות הבוט

**למשתמשים:**
- `/start` - הרשמה
- `/browse` - גלוש בפרופילים

**לאדמין בלבד:**
- `/admin` - פאנל ניהול מלא

## מה הבוט עושה

- ✅ הרשמה עם אימות גיל (נשים 40+, גברים מתחת ל-40)
- ✅ אימות זהות עם צילום תז
- ✅ פאנל אדמין לאישור/דחייה/חסימה
- ✅ מערכת לייקים והתאמות
- ✅ ערעור על חסימה
- ✅ תז נשמר עד שהאדמין מוחק אותו ידנית
